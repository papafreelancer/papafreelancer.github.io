Базовая информация по HTML5

Стартовый шаблон для HTML документа

  • <!DOCTYPE html> - данный тег сообщает браузеру (и всем остальным) какая версия HTML
  • <html lang="en"> - главный родитель всех тегов на странице
  • <head> - цель данного тега - хранение элементов, невидиммых для пользователя, но полезных для браузера элементов
  • <meta charset="UTF-8"> - указывает кодировку документа
  • <meta name="viewport"> - позволяет веб-разработчику контролировать размер окна просмотра
  • <body> - используется для хранения контента, отображаемого в окне браузера
